portfolio

Seam-carving program

Course Project @ University of Toronto (ESC190 Data Structures & Algorithms). With Chaewon L.

This python program uses dynamic programming to identify vertical paths of pixels in an image that can be removed without distorting the image. It can continue to remove these less important paths one by one, gradually making the image less wide. For more details, please refer to the GitHub repository.

MATLAB Script that predicts the failure load of a specified bridge

Final Project @ University of Toronto (CIV102 Structures and Materials). With Christine L. and Jocelyn Q.

This MATLAB script calculates the factor of safety for various bridge designs given a load of 400 N. It creates an envelope shear force and considers various types of failure including plate buckling, flexural compression, shear force in the glue and in the MATboard of the bridge, etc. This code was used to build an 850 g bridge out of matboard that supported a live load of 1000 N (100 times greater than its self-weight). This result ranked us third place in the Engineering Science 2T6 class. To access the code, please refer to its GitHub repository

Synonym Detector

Course Project @ University of Toronto (ESC180 Introdction to Computer Programming). With Chaewon L.

This python program draws from data in two books and identifies the relationship between words to then identify the synonym of a given word from a list of options. We were able to obtain a 70% accuracy. To avoid timing out due to large amounts of data, it has a low time complexity of O(n). For more details, please refer to the GitHub repository
