about me

portfolio

A selection of projects that blend software, systems thinking, and applied engineering.

SmartStudy: Personalized study habit recommender

Course Project @ University of Toronto (ECE324 Machine Intelligence Applications). Team: StudyArchitects.

A machine learning tool that takes a student's current study habits and a target GPA, then recommends the minimal lifestyle changes needed to get there. The pipeline combines a TabPFN GPA predictor, a numerical optimizer, KNN peer matching, and GPT-4 text generation, all wrapped in a Gradio web app. View the full project page →

PANDA-POSSUM: Canoe generator and optimizer

University of Toronto Concrete Canoe Team. Hull & Structural Members: Albert, Elorie, Folu, John, Lucas, Elliot.

A Python-based hull generation and analysis workflow that evaluates canoe geometry across race loadcases, computes hydrodynamic and stability metrics, and scores each variant against weighted target values. The bulk mode runs parameter sweeps in parallel, then POSSUM narrows the next search window using golden-ratio style updates. View the full project page →

Automated water sanitation system

Course Project @ University of Toronto (ESC204 Praxis III). With Jessica M., Hyunji K., Preetinder S., Kristen M., Seoyun Y., Kumkum C., and Joaquin C.

With the use of 3D printing, sensors, motors, and online tools such as Fusion360 and Notion, my team and I prototyped a system that could heat water to a specified temperature, for a given time, and then automatically dispense it to a tank of potable water. View the full project page →

Seam-carving program

Course Project @ University of Toronto (ESC190 Data Structures & Algorithms). With Chaewon L.

This C program uses dynamic programming to identify vertical paths of pixels in an image that can be removed without distorting the image. It can continue to remove these less important paths one by one, gradually making the image less wide without distorting it or removing meaningful aspects of it. For more details, please refer to the GitHub repository.

MATLAB Script that predicts the failure load of a specified bridge

Final Project @ University of Toronto (CIV102 Structures and Materials). With Christine L. and Jocelyn Q.

This MATLAB script calculates the factor of safety for various bridge designs given a load of 400 N. It creates an envelope shear force and considers various types of failure including plate buckling, flexural compression, shear force in the glue and in the MATboard of the bridge, etc. This code was used to build an 850 g bridge out of matboard that supported a live load of 1000 N (100 times greater than its self-weight). This result ranked us third place in the Engineering Science 2T6 class. For more details, please refer to its GitHub repository and the engineering drawings.

Synonym Detector

Course Project @ University of Toronto (ESC180 Introduction to Computer Programming). With Chaewon L.

This python program draws from data in two books and identifies the relationship between words to then identify the synonym of a given word from a list of options in linear time. We were able to obtain a 70% accuracy. For more details, please refer to the GitHub repository
