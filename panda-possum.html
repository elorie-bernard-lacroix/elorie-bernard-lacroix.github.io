PANDA-POSSUM:
Canoe Generator and Optimizer

Python · Hydrodynamics · Stability Analysis · Parallel Processing · Parametric Design · Iterative Optimization

View on GitHub

Project Overview

PANDA (Program for Automated Naval Design Analysis) is a Python workflow used by the University of Toronto Concrete Canoe team to generate and analyze canoe hull variants. The model takes geometric inputs such as length, paddler-box length, rocker, flare angle, waterline smoothing parameters, and cross-section shape parameters, then computes hydrostatic and performance outputs for each candidate design.

For each loadcase, the analysis pipeline evaluates metrics including freeboard, drag, stability tangent, leak angle, prismatic coefficient (Cp), and waterplane properties. Bulk mode sweeps across candidate parameter combinations and runs each variant in parallel across CPU cores.

The companion script POSSUM (PANDA Optimization Supplemental Script Utilizing Machine-learning) then reads the top-scoring variants and narrows parameter ranges using a golden-ratio style bisection update, enabling fast iterative search toward better-performing hulls over multiple runs.

How It Works

01

Define Inputs

Set parameter ranges and loadcases in the Excel input sheet, including paddler data and weighted target metrics.

02

Generate Variants

Create the full Cartesian product of canoe dimensions from the configured min/max ranges and iteration counts.

03

Analyze Hulls

Run hydrostatic, resistance, and stability analysis per variant and per loadcase through the Canoe and Loadcase models.

04

Score Results

Apply weighted Gaussian scoring against target values and standard deviations, then rank designs by average score.

05

Iterate with POSSUM

Use the best-performing hull from output CSVs to tighten search bounds and launch the next optimization cycle.

Core Outputs

Hydrostatics

Volume displaced, hull weight, center of mass, freeboard, and waterline-related geometry values.

Resistance

Drag from friction and wave-making components using the KAPER-based formulation.

Stability

Cross-curve tangent, tipping/leak behavior, and second-moment-based waterplane stability indicators.

Optimization Fitness

Per-loadcase weighted scores and overall average score out of 10 for ranking candidate designs.
