Project Overview

Students frequently struggle to balance their extracurricular and academic obligations. Although there exist online study tools and general recommendations, many of these resources lack personalization. SmartStudy — built by team StudyArchitects for ECE324 — is a machine learning tool that suggests individualized academic habits: the optimal weekly study hours, recommended tutoring time, recommended lecture attendance, and whether to engage in sports, music, volunteering, and other extracurricular activities.

The Students Performance Dataset (2392 high school students labeled with GPA) was used for training. A random forest regression model served as the baseline, paired with a coordinate descent algorithm that optimizes study habits with respect to predicted GPA. Iterations were compared using MSE and MAE on a 70/15/15 split. The most successful configuration was TabPFN with Bayesian optimization.

The final package also includes a k-nearest neighbors search that surfaces real students with similar profiles who already achieve the target GPA — offering concrete inspiration. GPT-4 then interprets the model's output into a clear, motivational action plan. The full source code, dataset, and demo are available on the GitHub repository.

How It Works

01

Input

The student enters their current habits — study hours, absences, activities — alongside a target GPA into the Gradio interface.

02

Predict

A TabPFN model trained on a student performance dataset estimates the GPA expected from the student's current habits.

03

Optimize

A numerical optimizer searches for the smallest set of habit changes that would push the predicted GPA to the student's target.

04

Match

KNN retrieves real students from the dataset with similar demographic profiles who already achieve the target GPA.

05

Recommend

GPT-4 takes the optimized habits and peer examples and generates a clear, motivational, plain-language action plan for the student.
